# # はじめに

Balance Bot (BB)は、暗号コインのポートフォリオのバランスを簡単にとることができます。最新のコイン価格を継続的にチェックし、ポートフォリオのバランスを保つためにコインの数を調整します。BBは24時間365日、完全に自動化されています。また、特定のコインのバランスを調整する必要がある場合には、手動で取引を開始することもできます。

BBアプリケーションは無料で提供されています。BBアプリケーションは無料で提供されており、ローカルのPC、クラウド、Raspberry Pi、高性能なNASなどにインストールすることができます。

## ご利用の前に

BBを使用するには、取引プラットフォームの口座が必要で、そこで暗号通貨を購入して暗号ポートフォリオを作成します。BBは現在、世界有数の暗号取引プラットフォームである[Binance account](https://www.binance.com/en/register?ref=121753644)でのみ動作します。まだBinanceアカウントをお持ちでない方は、こちらの[リンク](https://www.binance.com/en/register?ref=121753644)からアカウントを作成していただくと、取引手数料が10%割引になります。BBは24時間365日、売買注文を執行しますので、どんな割引でも価値があります。そして...このリンクを使うことで、BBプロジェクトをサポートすることになります。

取引コストをさらに削減するために、BNB（Binance crypto coin）を口座に残しておき、取引手数料の支払いに使用することをお勧めします。そうすることで、Binanceは取引ごとにさらに25％の割引をしてくれます。Binance [Fee Schedule](https://www.binance.com/en/fee/schedule)のページでこの機能が有効になっていることを確認してください。

BNBをあなたのバランスポートフォリオの一部にする必要はありませんが、そうすることは良いアイデアかもしれません。こうすることで、取引手数料を支払い、追加の割引を受けるのに十分なBNBを常に確保することができます。

BBは、ビットコイン（BTC）に対するアルトコインの取引を実行します。つまり、あなたのポートフォリオがあるアルトコインを多く保有している場合、そのアルトコインはBTCで売却され、その逆もまた然りです。したがって、BTCは常にポートフォリオの一部である必要があります。そして、BBを使えるのは、BTCと取引できるアルトコインだけです。しかし、これが制限になることはほとんどありません。

最後に、Binanceは0.0001BTCの最小注文サイズを適用しています。そのため、ポートフォリオの総量が少なすぎないように、そして/または、あまりにも多くの異なるコインに分けないようにしてください。そうしないと、取引が拒否される可能性があり、ポートフォリオのバランスを保つのが難しくなります。

## インストール

BB は Web アプリケーションであり、PC 上でローカルに動作させるためには、Node.js をインストー ルする必要があります。まずはダウンロードしてみましょう。

ダウンロードしたパッケージを選択し、インストールの手順に従います。標準の設定を受け入れるだけです。何も変更する必要はありません。

### Balance Botのインストール

BBにはインストーラーが付属しており、これをダウンロードしてPCに保存する必要があります。このインストーラーを使って、実際のBBアプリケーションをダウンロードし、インストールすることができます。また、このインストーラーを使って、今後のBBのアップデートを簡単にインストールすることができます。

以下の手順に従ってください。

1. BBインストーラーをダウンロードする[ここ](https://github.com/hodlerhacks/balance/archive/refs/heads/bot.zip)
2. ファイルを解凍し、balance-botフォルダをアクセスしやすい場所にコピー/移動します。ドライブのルートに置くことをお勧めします。その場合、インストーラーは「c:˶‾᷄д‾᷅˵」に置かれます。
3. 以下のプログラムを開きます。
   - Windowsの場合。コマンドプロンプト
   - macOSの場合：ターミナル
4. インストーラを保存したフォルダ（例：c:˶ˆ꒳ˆ˵）に移動します。
   - Windows の場合：「cd\」と入力して、ドライブのルートに移動します。cd balance-bot`と入力すると、正しいフォルダに入ります。
   - macOSの場合：「cd ~」と入力して、ドライブのルートに移動します。cd balance-bot`と入力して、正しいフォルダに入ります。
5. 5. `node install` と入力して、インストーラを起動します。
6. プロセスが完了するのを待ちます。
7. BBがあなたのシステムにインストールされました。

よくできました、難しい部分はこれで終わりです。 

### バランスボットの起動


BBを起動するには、同じフォルダ（例：`c:˶ˆ꒳ˆ˵`）から、`node bb`と入力します。

新しいボットが開始されたというメッセージが表示され、続いてURL: http://localhost:3001 が表示されます。このURLをブラウザで開くと、ウェブダッシュボードにアクセスできます。

これで完了です！バランシングの冒険を始める準備が整いました。

### ボットの設定

アプリケーションを起動すると、設定画面が表示されます。ボットを起動するには4つのステップが必要です。

1. 1. **Binance APIの詳細を入力**。Binance APIを作成したことがない場合は、[こちら](https://www.binance.com/en/support/faq/360002502072)を参考にしてください。
2. 2. **コイン設定を調整する**。BBはBinance APIを使用してお客様の取引口座にあるコインを確認します。このステップでは、利用可能なすべてのコインが表示されます。ここで、バランシングから除外したいコインを選択することができます。残りのコインについては、スライダを使ってどの配分でバランスを取るかを指定できます。デフォルトでは、すべてのコインが均等に配分されます。これでよければ、ここで何も変更する必要はありません。
3. 3. **スタートバランスの設定**。今からバランス調整を始める場合は、このステップを省略することもできます。しかし、すでに手動で、あるいは古いBBバージョンを使ってバランス調整を行っている場合は、ここで開始時のバランスを設定することができます。開始時の残高は、バランシングを始めたときのポートフォリオの各コインの価値を表しています。この情報はポートフォリオのパフォーマンスを計算するために使用されますので、少し時間をかけて適切に設定する価値があります。設定方法のヒントは、設定画面の(i)アイコンにマウスを合わせてください。
4. **一般的な設定を行います。最後に、ボットに名前を付け、コインのバランスを取る必要のある閾値を設定します。この閾値は、理想的な分布からの乖離率です。デフォルトでは3%に設定されています。例えば、10枚のコインがあり、それぞれのコインの目標分布が10％だとすると、BBは実際の分布が10.3％以上または9.7％以下になったときにコインの販売/購入を開始します。もしBBをクラウドで運用していて、異なるタイムゾーンのサーバーで運用している場合は、ここで時間を調整することができ、タイムスタンプはローカルタイムで表示されます。最後に、ここではBBを自動で動作させるか、マニュアルモードで使用するかを指定します。手動モードで始めることをお勧めします。そうすれば、すべての設定が完了し、期待通りに動作することを確認できます。慣れてきたら、この設定を自動モードに変更して、BBの動作に身を任せてみましょう。

## バランス・ボットの使用

BBアプリケーションにはいくつかの画面があり、以下に詳細を説明します。

### ポートフォリオ

これがメイン画面で、ポートフォリオと各コインの主要データが表になっています。現在の分配金と目標の分配金が表示されます。両者の相対的なデルタが設定された閾値を超えると、最後の列に赤の「売り」または緑の「買い」のインジケータが表示されます。BBをマニュアルモードで使用している場合は、テーブルの行を選択し、コインのバランスを取り戻すためにBBが注文を実行することを確認することができます。

この画面には、いくつかの重要なパフォーマンス指標も表示されます。

1. BTCでのポートフォリオの価値と、開始時の残高との比較
2. ユーロまたは米ドルでのポートフォリオの価値（左下のトグルに応じて）と、開始時の残高との比較。
3. BTCでの純増減とパーセンテージ。これらの値は、バランシング効果の指標です。バランシングを行わず、最初の購入後にコインをそのまま維持した場合のポートフォリオの価値と比較しています。
4. 正味の変化をユーロまたは米ドルで、またパーセンテージで表示します。ここではポジティブな数字が表示されていることを期待しましょう :wink:

### パフォーマンス

この画面では、上記で説明した主要なパフォーマンス指標が時間の経過とともにどのように変化したかを示す4つのグラフが表示されます。

### コイン分析

この画面では、ポートフォリオに含まれるコインの開始日からの相対的な価格変化のグラフが表示されます。この画面では、どのコインが勝ち組で、どのコインが負け組なのかが一目でわかります。

### イベント履歴

すべての注文は BB によって記録され、この画面の表では過去にさかのぼって注文の履歴を確認することができます。起こりうるエラーやその他のシステムアラートもここに含まれます。表の上にある2つのボタンを使って、どのイベントを表示するかをコントロールできます：オーダーやアラートです。filter events」では、特定のテキストを含むイベントを表示することができます。例えば、「BNB」と入力すると、すべての BNB の取引が表示されます。

### マルチインスタンス

BB を使って複数のボットを並行して動作させることができます。新しいボットを設定するには、コマンドプロンプト(Windows)またはターミナル(macOS)を開き、BBフォルダ(例：c:˶‾᷄д‾᷅˵)に入って、`node bb new`と入力します。既存のボットは `node bb` と同様に開始されますが、新しいボットが開始されるのもわかります。これが2番目のボットであれば、IDは002で、指定されたURL: http://localhost:3002 からアクセスできます。ここで、新しいボットを設定することができます。最初のボットは、http://localhost:3001 からアクセスできます。このようにして、それぞれのボットにブラウザの個別のタブでアクセスし、簡単に切り替えられるようになっています。

## よくある質問(FAQ)

##### 自分のポートフォリオには何枚のコインを保有できますか？

特に制限はありません。唯一の留意点は、Binanceが0.0001BTCの最小注文サイズを適用することです。したがって、ポートフォリオの総量が少なすぎないように、また、あまり多くの異なるコインに分けないようにしてください。そうしないと、取引が拒否される可能性があり、ポートフォリオのバランスを保つのが難しくなります。

##### 安定したコインとEURはサポートされていますか？

簡単に言うと、「はい！」です。BBはBinanceのあらゆる.../BTCとBTC/...のペアをサポートしています。安定したコインとEURは通常、アルトコインのように.../BTCペアを持ちません。しかし、BBでは代わりに同等のBTC/...ペアを自動的に使用して取引を行います。この場合、ポートフォリオ画面の表では、注文側の列に「reverse」と表示されます（BTC/...ペアを介して売り注文が買い注文に反転し、その逆も同様です）。

##### トレードが実行されていない

PCでBBを起動している場合は、電源が切れないように注意してください。

いずれにしても、市場の動き、特にお客様のコインの動きによっては、数時間の間にあまり取引が成立しないことがよくあります。もちろん、これは設定した閾値の割合に大きく依存します。

ボットがまだ稼働しているかどうか気になる場合は、ウェブダッシュボードを開いて、右上のステータスを確認してください。接続中」と表示され、最近のタイムスタンプが表示されているはずです。

##### ポートフォリオのテーブルはどのくらいの頻度で更新されますか？

更新頻度は10秒です。

##### 取引ツールでは異なる分布が表示されています 

トレーディングツールでは、お客様のポートフォリオの分布を表示することがよくあります。しかし、このデータはBBのように頻繁には更新されません。そのため、2つのアプリケーションの間に（大きな）違いが見られることがあります。疑わしい場合は、取引ツールのリアルタイムの価格情報をチェックし、BBの価格と比較するとよいでしょう。ほぼ同じ価格が表示されるはずです。

##### BTC残高が減っているのに、BBが注文を出さない

お客様のBTCが売られたり買われたりすることはありません。BTCは他のコインとの交換通貨として使用されます。そのため、設定した閾値を超えたBTCの分配が行われることがあります。その後のアルトコインの取引では、BTCの配分は自動的に修正されますので、問題ありません。
